Flujo de Datos

1. Inicio
   └─> select_otros_activos(account="B0000002") → lista símbolos
   └─> Crear TradingBotSpot por cada símbolo
   └─> Iniciar WebSocket klines

2. WebSocket recibe vela cerrada
   └─> BotManager.on_market_data(symbol, candle)
   └─> TradingBotSpot.on_market_data(candle)
   └─> TradingBotSpot.evaluate() → BUY/SELL/HOLD
   └─> Si BUY/SELL → ejecutar orden via MySpot

3. UI Update (cada 1 segundo)
   └─> Para cada widget:
       └─> bot.get_public_state()
       └─> bot.get_indicators()
       └─> widget.update(data)
